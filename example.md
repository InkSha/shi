```txt
盖闻天地之间，有数字曰：年纪，其值为四十有二。
盖闻天地之间，有文言曰：名称，其值为张三。
盖闻天地之间，有是非曰：活着，其值为是。
盖闻天地之间，有集合曰：五行，其值有金、木、水、火、土。
盖闻天地之间，有题解曰：人物，其有题姓，解张；题名，解三。

夫两数之和者，得数字之甲、数字之乙。
  还与天地以甲加之乙。
此终矣。

// int fb(int n)
夫斐波那契数列者，得数字之甲。
// if n <= 1 { return n }

  // var n1 = fb(n-1)
  盖闻天地之间，有数字曰：甲一，其值为斐波那契数列，传之甲减一。
  // var n2 = fb(n-2)
  盖闻天地之间，有数字曰：甲二，其值为斐波那契数列，传之甲减二。
  // return add(n1，n2)
  还与天地以两数之和，传之甲一，甲二。
此终矣。

  其可发声，需传之文言之言语，还与天地以文言。

  // public String 名讳

    将其姓氏取作姓。
    将其名讳取作名。

    若未有言语，则还与天地以无言。

  此终矣。

      其姓氏，其名讳，吃了，食物。
  此终矣。

  其身可移动，得数字之距离，无果与天地。
    其身发声，传之文言：
      其姓氏，其名讳，走了，距离，步。
  此终矣

此终矣。
```
